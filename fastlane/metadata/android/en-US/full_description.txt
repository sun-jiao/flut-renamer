<p><i>Renamer</i> is a powerful yet easy-to-use tool designed to help users effortlessly manage and rename files and directories. No more manually changing file names one by one – our app offers various features including inserting text, inserting file metadata and Exif data, replacing text, deleting text, rearranging, and more, allowing you to quickly batch rename files according to your needs.<br> </p><ul><li>Multiple renaming options: Easily achieve batch file renaming by inserting, replacing, deleting text, and rearranging.</li><li>Insert file metadata and Exif data: Extract information from file metadata and Exif data and insert it into the file name.</li><li>Completely open-source and free: Our app is entirely open-source and contains no advertisements or in-app purchases, allowing you to use it freely at any time.</li><li>Cross-platform compatibility: Built on the Flutter framework, our app can run on multiple operating systems, enabling you to use it anytime, anywhere.</li></ul>
